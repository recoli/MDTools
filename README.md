# MDTools

### amber_to_gmx_by_XL: 

+ Convert AMBER input files (prmcrd, prmtop) to GROMACS format (gro, top)

    > Usage: 
    
        perl amber_to_gmx_by_XL.pl 
            -top [name_of_amber_top_file] 
            -crd [name_of_amber_crd_file] 
            -name [name_of_your_molecule] 
            -glycam [yes_or_no]
